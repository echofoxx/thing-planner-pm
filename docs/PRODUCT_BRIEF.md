# Product Brief: Thing Planner PM

## Vision

Thing Planner PM is a simple, flexible, template-first project management app for professional and personal projects. It gives users enough structure to manage real work without the friction of enterprise project tools.

## Target users

- Individual project owners
- Small teams
- Home/auto/project hobby users
- Office teams that need lightweight structure
- Standards/document project leads
- Software prototype builders

## Differentiator

The app combines:

- Trello-like Kanban simplicity
- Excel lifecycle planning familiarity
- WBS-style task traceability
- Lightweight reporting
- Completed project reuse
- AI-assisted planning

## MVP success criteria

The MVP is successful when a user can:

1. Create a project from a template.
2. See phases, tasks, risks, and a starter workflow.
3. Drag work through a Kanban board.
4. See dashboard metrics update automatically.
5. Review WBS/task details.
6. Generate a status report.
7. Export/import JSON.
8. Export CSV.
9. Close and archive the project.
10. Reuse the completed project as a future template.
